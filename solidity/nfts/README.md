## Deploying

### Prerequisites
create a `.env` file and include the following environment variables
```
HARDHAT_RINKEBY_URL=ADD_APP_URL_HERE
HARDHAT_RINKEBY_ACCOUNT=ADD_WALLET_KEY_HERE
```
### Testnet
```
npx hardhat run scripts/deploy.js --network rinkeby
```
